BitShares Blockchain Service
=============================

Public Blockchain service for BitShares network, goals for this project:

1.  Public Blockchain Data API
2.  Integrate with light-weight client
3.  Public Blockchain Explorer

Install
--------

    bundle install
  
Test
-----

    rspec
    
Branch
-------

`master` branch contains productional stable code, will be deployed to production server  
`develop` branch contains latest development code, will be deployed to staging server
